Operators in Java





What is Operators in Java?


In Java, operators are symbols that perform operations on operands. Operands can be variables, constants, or expressions. Java supports a variety of operators, which can be categorized into the following types:


1. Arithmetic Operators:

  • + (addition)
  • - (subtraction)
  • * (multiplication)
  • / (division)
  • % (modulo, returns the remainder)
        
            int a = 10;
            int b = 3;
            int sum = a + b;    // 13
            int difference = a - b;  // 7
            int product = a * b;  // 30
            int quotient = a / b;  // 3
            int remainder = a % b;  // 1

2. Relational Operators:

  • == (Equal to)
  • != (Not Equal to)
  • > (Less then)
  • < (Grater than)
  • >= (less than equal to)
  • <= (grater then equal to)
      
        int x = 5, y = 8;
boolean isEqual = (x == y);    // false
boolean isNotEqual = (x != y); // true
boolean greaterThan = (x > y); // false
boolean lessThan = (x < y);    // true
